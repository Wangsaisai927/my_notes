## 故事
1. 现有的研究关于机器人的研究都是在实验室里的，只分析特定的场景和特定的机器人，甚至指令也是特定的。
2. 以前的研究都是落在机器人怎么更懂人，这篇文章更多落在对“人”行为模式的研究：人的行为往后是“做出这个行为是要表达什么？” / 往前是“在什么样的对话角色（文章抽象成社交场景，这样更体现泛性）中人是否会做出不同的信息传递行为？
3. 机器人的外形（文中提出类人型 / 动物型 / 地板流 / 天空型）会对人传递指令/信息产生影响。

## 主要实验
1. 定义了什么是社交场景 / 什么样的机器人是社交机器人。
2. 机器人的形象 / 人在社交场景中的角色 / 人要做出的社交动作三者定义为实验的自变量，具体实施为：机器人有四种形象 / 人在对话场景中有讲话人和听话人的区分，机器人扮演的是服务员 / 人要传达出13种意图，具体怎么传达可以自己决定（这里的单词叫elicictation）。
3. 其实机器人是虚拟（看不见摸不着）的，采取绿野仙踪控制法。
4. 机器人会出现送饮料 / 问问题 /打断 / 出错等行为。
5. 24（人）×2（两种对话角色）×13（传递的信号种类）=624的数据量。 
## 实验结论和可以沿着做的内容
1. 人类的表达方式不统一，用非语言信号交流居多。
2. 人的表达行为受角色影响。
3. 机器人的外形会影响人的表达行为。
4. 不同的任务有典型的表达模式。
